Best Time to Buy and Sell Stock - Java Solution

1. Problem Statement

You are given an array prices where prices[i] is the price of a stock on the ith day. You want to maximize your profit by choosing a single day to buy one stock and another day to sell it. Return the maximum profit. If no profit is possible, return 0.

Example 1:

Input: prices = [7, 1, 5, 3, 6, 4]
Output: 5
Explanation: Buy on day 2 (price = 1) and sell on day 5 (price = 6), profit = 6-1 = 5.

Example 2:

Input: prices = [7, 6, 4, 3, 1]
Output: 0
Explanation: No profit is possible as prices keep decreasing.

2. Algorithm Used

The algorithm uses a two-pointer technique to track the best day to buy and sell stocks for maximum profit. The "buy" pointer points to the lowest price and the "sell" pointer iterates over future days to calculate the profit.

3. Step-by-Step Explanation with Visualization

Example: Input Array: [7, 1, 5, 3, 6, 4]

Step 1: Initialize buy = 0, sell = 1, maxProfit = 0.

Iteration 1:

Array: [ 7, 1, 5, 3, 6, 4 ]

Compare prices[0] (7) and prices[1] (1). Since 1 < 7, update buy = 1.

Iteration 2:

Array: [ 7, 1, 5, 3, 6, 4 ]

Compare prices[1] (1) and prices[2] (5). Profit = 5 - 1 = 4. Update maxProfit = 4.

Iteration 3:

Array: [ 7, 1, 5, 3, 6, 4 ]

Compare prices[1] (1) and prices[3] (3). Profit = 3 - 1 = 2. maxProfit remains 4.

Iteration 4:

Array: [ 7, 1, 5, 3, 6, 4 ]

Compare prices[1] (1) and prices[4] (6). Profit = 6 - 1 = 5. Update maxProfit = 5.

Iteration 5:

Array: [ 7, 1, 5, 3, 6, 4 ]

Compare prices[1] (1) and prices[5] (4). Profit = 4 - 1 = 3. maxProfit remains 5.

Final Result: The maximum profit is 5.

4. Time and Space Complexity

Time Complexity: O(n) – Traverses the array once, making it linear time.

Space Complexity: O(1) – Uses only a few extra variables, making it constant space.

5. Conclusion

The two-pointer technique is an efficient approach to solving the stock profit problem, achieving O(n) time complexity. This solution guarantees an optimal result for any valid input within the given constraints.

6. Java Code


// Stock Profit Maximization Solution
class Solution {
    public int maxProfit(int[] prices) {
        int buy = 0;
        int sell = 1;
        int maxProfit = 0;
        int currentProfit = 0;
        
        while(sell < prices.length) {
            if(prices[buy] > prices[sell]) {
                buy = sell;
            } else {
                currentProfit = prices[sell] - prices[buy];
                if(maxProfit < currentProfit) {
                    maxProfit = currentProfit;
                }
            }
            sell++;
        }
        return maxProfit;
    }
}
