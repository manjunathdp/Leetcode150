Rotate Array - Java Solution

1. Problem Statement

Given an array nums, rotate the array to the right by k steps, where k is non-negative.

Example 1:

Input: nums = [1,2,3,4,5,6,7], k = 3
Output: [5,6,7,1,2,3,4]

Example 2:

Input: nums = [-1,-100,3,99], k = 2
Output: [3,99,-1,-100]

2. Algorithm: Array Reversal

This approach uses three steps:

  1. Reverse the entire array.
  2. Reverse the first k elements.
  3. Reverse the remaining n - k elements.

3. Step-by-Step Explanation with Visualization

Example 1: Input Array: [1,2,3,4,5,6,7], k = 3

Array: [ 7, 6, 5, 4, 3, 2, 1 ]

Step 2: Reverse the first 3 elements

Array: [ 5, 6, 7, 4, 3, 2, 1 ]

Step 3: Reverse the remaining 4 elements

Array: [ 5, 6, 7, 1, 2, 3, 4 ]

Final Result: The rotated array is [5, 6, 7, 1, 2, 3, 4]

4. Time and Space Complexity

Time Complexity: O(n), where n is the length of the array.

Space Complexity: O(1), since all operations are performed in-place.

5. Conclusion

The three-step reversal method is an efficient solution to rotate an array. It uses O(n) time complexity and performs the operations in-place using O(1) space.

6. Java Code


// Rotate Array Solution (Array Reversal)
class Solution {
    public void rotate(int[] nums, int k) {
        k = k % nums.length;
        rotate(nums, 0, nums.length - 1);
        rotate(nums, 0, k - 1);
        rotate(nums, k, nums.length - 1);
    }

    private void rotate(int[] nums, int start, int end) {
        while (start < end) {
            int temp = nums[start];
            nums[start] = nums[end];
            nums[end] = temp;
            start++;
            end--;
        }
    }
}
